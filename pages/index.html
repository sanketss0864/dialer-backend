Twilio VoIP Server

Available endpoints:

  • POST /token/generate - Generate access token
  • POST /voice/incoming - Handle incoming calls
  • POST /voice/outbound - Make outbound calls
  • GET /voice/recordings - List recordings
  • GET /logs/calls - Get call logs
  • GET /logs/calls/:callSid - Get specific call details
  • GET /logs/transcriptions - Get all transcriptions
  • GET /logs/transcriptions/:transcriptionSid - Get specific transcription
